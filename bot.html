Alliance Setup

Connect your guild to the network in 3 simple steps.

1

Invite the Bot

Add Depths Saviors to your server. It requires permissions to view channels and send messages.

Invite Link
2

Request Approval

For security, all guilds must be manually whitelisted. Send your server invite link to an admin via Discord:

@Rindw (Co-Owner)
@Zako_tb (Owner)
3

Configure Settings

Once approved, run these commands in your server to finalize setup.

/setchannel #alerts Select the text channel where gank alerts will appear.
/setgankping @Role Choose the role to ping when a rescue is needed.
/setallyrole @Role Authorize specific members to call ganks (Admins can always call).

Commands not working? Check /gankhelp or contact support.
